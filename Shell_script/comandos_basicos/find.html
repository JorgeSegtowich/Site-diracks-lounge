find
Encontra arquivos procurando pelo nome, data, tamanho e outras propriedades

 
Sintaxe
        find pasta [Opção] [propriedade]

 
Parâmetros
       
 
pasta 		Endereço do diretório a ser pesquisado
 
propriedade 		Nome, data, tamanho do arquivo
 
Opção -name 		Especifica o nome do arquivo ou parte dele
 
Opção -iname 		Ignora a diferença entre maiúsculas e minúsculas no nome
 
Opção-type		Especifica o tipo do arquivo (f=arquivo, d=diretório, l-link)
 
Opção -mtime 		Mostra os arquivos modificados a N dias
 
Opção-size		Mostra os arquivos que possuem um tamanho especificado
 
Opção -user 		Mostra os arquivos de determinado usuário
 
Opção-exec		Executa comandos com os arquivos encontrados
 
Opção -ok 		Executa comandos com os arquivos encontrados, com confirmação
 
Opção-not		Inverte a lógica da expressão

 
Exemplos e dicas
        $ find /etc -iname 'x*'
        /etc/X11
/etc/xinetd.d
/etc/log.d/conf/logfiles/xferlog.conf
...
        *Para procurar mais de um tipo de arquivo:
        $ find . -name '*.txt' -or -name '*.html'
