mv
Mover arquivos e diretórios

 
Sintaxe
        mv endereco_original endereco_novo

 
Parâmetros
       
 
endereco_original 		Endereço original do arquivo que deseja mover o arquivo, exemplo /home/user/arquivo.txt
 
endereco_novo 		Endereço do diretório para o qual deseja mover o arquivo, exemplo /home/user/bin (mover arquivo.txt para a pasta bin)
