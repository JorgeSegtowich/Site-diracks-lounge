rev
Inverte a ordem dos caracteres de uma mensagem. Funciona como um espelho, exemplo 'texto' vira 'otxet'. Deve ser utilizado com o comando echo

 
Sintaxe
        echo "o rev é muito bacana" | rev

 
Parâmetros
       

 
Exemplos e dicas
        $ echo "O rev é muito bacana." | rev
        .anacab otium é ver O
        $ echo "O rev é muito bacana." | rev | rev
        O rev é muito bacana.
        * Usando o cut sozinho, não há como especificar o último campo, ou o último caractere. Com o rev, isso é possível:
        $ echo "123456789" | rev | cut -c 1
        9
        * Similarmente, para pegar os últimos 5 caracteres:
        $ echo "123456789" | rev | cut -c 1-5
        56789
