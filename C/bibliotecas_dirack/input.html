Biblioteca de funções para tratar com entradas de parâmetros a partir da linha de comandos. Os parâmetros e seus valores são armazenados em uma tabela simbólica de parâmetros, inicializada pela função init(argc,argv) que recebe como argumentos as variáveis int argc e char* argv, que são os parâmetros passados à função main() na inicialização do programa

 
Estruturas
struct tb_pars Tabela de parâmetros
int tb_pars.tam Número total de parâmetros. Tamanho da tabela de parâmetros
char** tb_pars.pars Parâmetro armazenado na tabela de parâmetros
tabela Apelido (alias) da tabela de parâmetros

 
Funções
       
 void init(int tam, char **p) Inicializa a tabela simbólica de parâmetros
 bool getint(char *string, int *pi) Extrair um parâmetro do tipo int da tabela simbólica de parâmetros
 bool getfloat(char *string, float *pi) Extrair um parâmetro do tipo float da tabela simbólica de parâmetros
 bool getdouble(char *string, double *pi) Extrair um parâmetro do tipo double da tabela simbólica de parâmetros

 
Exemplos e dicas
        * Salve input.h na pasta /usr/include para poder utilizar a seguinte diretiva nos seus programas:
        #include <input.h>
       
        * Inicie a tabela de parâmetros no seu programa, antes de utilizar as funções. Basta incluir a linha abaixo:
        init(argc,argv);
       
        * As funções get são melhor utilizadas com condicionais ifnot. No exemplo abaixo, a função getfloat retornará 'true' se o parâmetro 'f' foi passado pela linha de comandos e atribuirá o valor passado à variável correspondente. Se 'f' não foi passado, getfloat retorna 'false' e atribui o valor 0 à 'f' definido como default:
        if (!getfloat("f",&f)) f=0.;
       
        * Da mesma forma getint:
        if (!getint("i",&i)) i=0.;
       
        * Idem para getdouble
        if (!getdouble("d",&d)) d=0.;
