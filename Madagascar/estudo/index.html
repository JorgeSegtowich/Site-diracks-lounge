Madagascar API's Tutoriais		 

Esta breve introdução à API do Madagascar tem o objetivo de ensinar como incluir um programa na base do Madagascar. Isto será feito através de um programa que será escrito em C, mas poderia ter sido escrito (com as devidas adaptações) com qualquer outra linguagem que a API do Ma- dagascar suporta: C, C ++ , Fortran 77 e 90, Python, Java e MATLAB * . O foco será dado à como entrar e sair com arquivos no formato RSF na- tivo ao Madagascar, através de suas funções. Para tal, será desenvolvido durante o minicurso um programa que calcula derivadas primeiras e se- gundas de um certo dado de entrada.
